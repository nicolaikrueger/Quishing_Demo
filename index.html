Quishing: Die Gefahr hinter QR-Codes

Manipulierte QR-Codes können Sie auf gefährliche Webseiten führen. Bleiben Sie wachsam!

Wie können Sie sich schützen?

  • Prüfen Sie, woher der QR-Code stammt.
  • Überprüfen Sie die URL, bevor Sie die Seite öffnen.
  • Stellen Sie sicher, dass die Quelle vertrauenswürdig ist.

Danke, dass Sie diesen QR-Code gescannt haben – denken Sie daran:

„Man sollte nie unbedacht QR-Codes scannen.“
